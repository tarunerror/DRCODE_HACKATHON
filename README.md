## 🛠️ Installation Guide  

Clone the repository:  
```sh
git clone https://github.com/tarunerror/DRCODE_HACKATHON.git
cd DRCODE_HACKATHON
```

### 🚀 Frontend Setup  
```sh
cd frontend  
npm install  
npm start  
```

### ⚙ Backend Setup  
```sh
cd backend  
npm install  
npm start  
```

Now, open your browser and enjoy the **DRCODE_HACKATHON experience!**
